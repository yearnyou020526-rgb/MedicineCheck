# 吃药打卡

这是一个极简 Android 小组件 App。它只在本地保存数据，不联网、不登录、不上传。

## 功能

- Android 桌面 1x1 Widget 小组件。
- 未打卡：红色背景，中间白色药片。
- 已打卡：绿色背景，药片淡化，中间白色对勾。
- 点击未打卡小组件：记录今天已吃药。
- 点击已打卡小组件：弹出确认提示，确认后取消今日打卡。
- 第二天刷新小组件时自动显示为未打卡。
- App 主界面包含药名设置、今天状态、最近 30 天记录。

## 打包

每次 push 到 GitHub 后，GitHub Actions 会自动构建 debug APK，并上传名为 `MedicineCheck.apk` 的 artifact。
